## 📋 À propos

Plateforme web de **covoiturage** permettant aux conducteurs de proposer des trajets et aux passagers de réserver des places.

### ✨ Fonctionnalités

- Inscription & Connexion des utilisateurs
- Publication et recherche de trajets
- Réservation de places en temps réel
- Gestion de profil conducteur/passager
- Interface responsive (mobile friendly)

---

## 🛠️ Technologies

- **Backend** : PHP (Framework CodeIgniter)
- **Base de données** : MySQL
- **Frontend** : Bootstrap 5, JavaScript
- **Architecture** : MVC
