## 📋 Description du Projet

**Convoiturage App** est une plateforme web permettant de gérer les trajets de covoiturage entre utilisateurs.  
Elle permet aux conducteurs de proposer des trajets et aux passagers de réserver des places.

### ✨ Fonctionnalités principales

- **Inscription & Connexion** des utilisateurs (Conducteurs & Passagers)
- **Publication de trajets** (départ, destination, date, heure, prix, places disponibles)
- **Recherche de trajets** avec filtres
- **Réservation de places** en temps réel
- **Gestion du profil** utilisateur
- **Tableau de bord** pour les conducteurs
- **Interface responsive** (mobile friendly)

---

## 🛠️ Technologies utilisées

- **Backend** : PHP (CodeIgniter Framework)
- **Base de données** : MySQL
- **Frontend** : HTML5, CSS3, Bootstrap 5, JavaScript
- **Architecture** : MVC (Model-View-Controller)
