## 📋 À propos

Plateforme web de covoiturage permettant aux conducteurs de proposer des trajets et aux passagers de réserver des places.

### ✨ Fonctionnalités

- Inscription & Connexion
- Publication et recherche de trajets
- Réservation de places
- Gestion de profil
- Interface responsive

---

## 🛠️ Technologies

- **Backend** : PHP (CodeIgniter)
- **Base de données** : MySQL
- **Frontend** : Bootstrap 5, JavaScript
- **Architecture** : MVC
